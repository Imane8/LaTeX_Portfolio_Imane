# Academic Thesis – LaTeX Template

This project presents a professional **LaTeX thesis structure** designed for long academic documents (Master theses).

The repository focuses on **document architecture and layout engineering**, not on scientific content.

## Key Skills Demonstrated
- Modular LaTeX project organization
- Professional page layout (geometry, headers, footers)
- Front matter management (cover, abstract, lists)
- Chapter structuring and numbering
- Bibliography and appendices integration
- Academic publishing standards

## Files
- `main.tex` – Core LaTeX structure and configuration  
- `Preview.pdf` – Compiled layout preview

## Purpose
This project demonstrates my ability to design, maintain, and structure complex LaTeX documents for academic and professional use.
